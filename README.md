# Vuejs+SSM实现前后端分离的网站
[项目demo](https://yeahqing.cn/reagents/)

### 前端: 用户界面的显示及路由
  - Vue Cli3
  - BootstrapVue
  - vue-router
  - vue-axios
  - mavon-editor
### 后端: api数据接口的提供
  - SpringMVC
  - Spring
  - Mybatis
  - Maven
  - PageHelper
  - jdk1.8
  - Mysql8
  - Servlet4.0
### 服务器: 运行tomcat

  - 腾讯云
  - centos7.4
